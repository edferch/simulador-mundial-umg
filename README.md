# 🏆 World Cup Logistics & Simulator UMG

Este es el proyecto oficial del simulador del mundial para la Universidad Mariano Gálvez (UMG). El sistema utiliza una arquitectura de Backend en Java y un Frontend web para el motor de reproducción visual de los partidos.

> ⚠️ **Requisitos Obligatorios del Sistema:**
> Para evitar errores de compatibilidad, todos los miembros del equipo deben tener instaladas exactamente estas versiones:
> * **Java (JDK):** Versión 25.0.1 (LTS)
> * **PostgreSQL:** Versión 16.x

## 🛠️ Tecnologías Utilizadas
* **Backend:** Java 25 + Spring Boot (JPA / Hibernate)
* **Base de Datos:** PostgreSQL 16
* **Frontend:** HTML, CSS, JavaScript
* **Entorno de Desarrollo:** VS Code

## 🚀 Instrucciones para el Equipo (Instalación y Configuración)

Para correr este proyecto en tu máquina local, sigue estos pasos al pie de la letra:

### 1. Preparar la Base de Datos
1. Abre **pgAdmin 4** y conéctate a tu servidor local.
2. Crea una base de datos vacía llamada exactamente **`mundial_umg`** (todo en minúsculas).
3. No es necesario crear tablas a mano, el código en Java (Hibernate/JPA) lo hará automáticamente al correr el proyecto.

### 2. Instalar extensiones
1. Instalar en VSCode las siguientes extensiones:
   Java de microsoft
   LiveServer
